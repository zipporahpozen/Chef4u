# Chef4U - Online Chef Registration and Ordering Platform

Welcome to Chef4U, an innovative platform that allows users to register, book private chefs for events, browse diverse menus, and rate both chefs and menus based on their order experience.

---

## Project Overview

Chef4U supports two main user roles:

* **Chefs:** Can create personal profiles, upload unique menus, manage bookings, and showcase their profiles to users.
* **Users:** Can register, browse chef profiles, book chefs for their events, view menus, and leave ratings and reviews for chefs and menus.

---

## Key Features

### Registration and Profile Management

* **Easy and quick registration** for both chefs and users.
* **Chef profile management:** Chefs can update personal info, upload photos, set availability, and manage menus.
* **User profile management:** Users can update personal details, view past orders, and manage preferences.

### Chef Booking System

* Users can select a chef from the available list and book them for an event.
* Scheduling system to specify date, time, and event details.
* Option to cancel or modify bookings.

### Menus Display and Management

* Chefs can create and upload detailed menus with dish descriptions, prices, and images.
* Users can browse menus before placing orders.
* Filters and search options by cuisine type, price, and more.

### Ratings and Reviews

* After each order, users can rate both the chef and the menu.
* Chef ratings are calculated based on the number of orders and quality of reviews.
* Menu ratings help users choose recommended and popular menus.
* Users can leave detailed feedback for improvements and compliments.

### Messaging and Notifications

* Automated notifications about booking status.
* Internal messaging system between users and chefs.

---

## Technologies Used

* Frontend: - HTML5 - CSS3 (including Flexbox/Grid) - Vanilla JavaScript (ES6+) 
* Backend: FastAPI / Node.js 
* Database:  MongoDB
* Authentication: JWT
* Image Storage: Firebase Storage

## Installation and Setup

1. **Download or Clone the Repository**  
   - Download the project ZIP file or clone the repository using the command:  
     ```bash
     git clone https://github.com/zipporahpozen/Chef4u.git
     ```

2. **Open the Project Folder**  
   - Navigate to the project folder on your computer.

3. **Run the Website**  
   - Open the `home.html` file in your preferred web browser (such as Chrome, Firefox, or Edge).

4. **No Additional Setup Required**  
   - This project uses only HTML, CSS, and JavaScript, so no server or additional installations are necessary.  
   - If you want to run a local server for development, you can use tools like the Live Server extension in Visual Studio Code.
